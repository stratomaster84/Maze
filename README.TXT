---------------------------------------------
Программа работает в двух режимах:
	1. Resolve Mode
	2. Maze Modify
---------------------------------------------
1. Режим "Resolve Mode"
	- Нажатие левой кнопки мыши по ячейке устанавливает точку старта
	- Нажатие правой кнопки мыши по ячейке устанавливает точку финиша
	- Нажатие кнопки "Resolve" ищет кратчайший путь по алгоритму ЛИ
	- Флаг "Allow Diagonal Step" разрешает/запрещает шаги по диагонали

2. Режим "Modify Mode"
	- В элементе "Maze Size" можно установить количество ячеек по ширине ("Width") и по длине ("Height")
	- Нажатие левой кнопки мыши по границе ячейки устанавливает/снимает препятствие
	- Нажатие кнопки "Generate Random Maze" генерирует простой случайный лабиринт
	- Нажатие кнопки "Generate Ellers Algorithm Maze" генерирует лабиринт по алгоритму Эллера (будет реализована позже)
